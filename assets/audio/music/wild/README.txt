野圖背景音樂 / Wild-map background music
=========================================

把 CC0 授權的音樂檔放進這個資料夾，進入任何野外地圖時會自動隨機播放。
Drop CC0-licensed music files here; they play automatically (randomly) on any wild map.

支援格式 / Supported formats: .ogg (建議/recommended), .mp3, .wav
- 多首檔案會輪流隨機播放；只有一首時會循環。
- Multiple files play in random rotation; a single file loops.

注意 / Notes:
- 用 Godot 編輯器開啟專案一次，讓音檔完成匯入 (import) 後才會生效。
- Open the project in the Godot editor once so the audio files get imported.
- 只放你有權使用的音樂（CC0 / public domain）。Only use music you are licensed to use (CC0 / public domain).
